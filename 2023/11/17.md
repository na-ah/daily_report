## わかったこと
- ブロックとして扱う際には引数に&をつける
- Procオブジェクトを作る方法は4つある
 - `Proc.new { |a, b| a + b}`
 - `proc { |a, b| a + b }`
 - `->(a, b) { a + b }`
 - `lambda { |a, b| a + b }`
- Proc.newと違い、ラムダは引数の個数に厳しい
